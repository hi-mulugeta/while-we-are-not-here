# **App Name**: MessageSlip

## Core Features:

- Form Input: Capture message details including recipient, sender, date, time, and notes.
- Status Selection: Select the appropriate status from a grid of checkboxes (Telephoned, Came to see you, etc.)
- Message Input: Input a detailed message in a multi-line text area.
- Print Function: Enable printing of the completed form in a clean, A4-friendly format.
- Form Validation: Perform basic form validation to ensure all required fields are completed.
- Clear Form: Provide a button to clear all fields and reset the form.
- PDF Export: Ability to save a completed form as a PDF document.

## Style Guidelines:

- Background color: Very light gray (#F5F5F5) to mimic paper.
- Primary color: Dark gray (#333333) for text and lines, offering a professional feel.
- Accent color: Slightly lighter gray (#808080) for subtle interactive elements, like hover states and checkbox borders.
- Headings font: 'Playfair', serif, for the “IMPORTANT MESSAGE” title and section headers.
- Body and labels font: 'Inter', sans-serif, for all other text to ensure readability.
- Use simple, subtle icons to enhance clarity where appropriate.
- Employ generous whitespace and thin borders/dividers to structure the form clearly.
- Use very subtle animations on hover/focus states for interactive elements.
